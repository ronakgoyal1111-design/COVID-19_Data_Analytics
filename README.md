# 🦠 COVID-19 Data Analysis using Python

## 📌 Overview
This project performs **Exploratory Data Analysis (EDA)** on COVID-19 datasets to understand the spread and impact of the pandemic across different regions. It focuses on extracting meaningful insights from real-world data using Python.

The analysis includes tracking confirmed cases, deaths, and recoveries, along with visualizing trends over time.

---

## 🎯 Objectives
- Analyze COVID-19 data to identify trends and patterns
- Compare country-wise performance
- Visualize growth of cases, deaths, and recoveries
- Derive actionable insights using data analytics

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📂 Dataset
The dataset contains real-world COVID-19 data including:
- Confirmed cases
- Deaths
- Recoveries
- Country/Region-wise information
- Date-wise progression

---

## 🔍 Key Steps Performed
1. **Data Collection**
   - Imported dataset from CSV files

2. **Data Cleaning**
   - Handled missing values
   - Removed irrelevant columns
   - Converted data types

3. **Exploratory Data Analysis (EDA)**
   - Analyzed trends over time
   - Compared different countries
   - Calculated growth rates

4. **Data Visualization**
   - Line plots for time-series analysis
   - Bar charts for country comparison
   - Heatmaps for correlation analysis
